# dbpage - a page to show database

## a flask project
This use flask to do show database with limit

### features
* show database records all columns
* show records with limit 100
* select by condition with parameters
* combination same parameters for 'or' query 

### statistic
* choose count unit by day, 00:00~24:00
* choose count unit by week, sun day 00:00~ sat dayy 24:00
* choose count unit by month, 1/1 00:00 ~ 1/31 24:00
